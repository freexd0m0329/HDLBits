# Getting Started
如果您的電路正確，您將會看到「狀態：成功！」還有其他幾種可能性：

- **編譯錯誤** — 電路無法編譯
- **模擬錯誤** — 電路編譯成功，但模擬未完成
- **不正確** — 電路編譯和模擬成功，但輸出與參考不符
- **成功！** — 電路正確

### Problem：

建立一個沒有輸入但有一個輸出的電路，該輸出應始終為1（或邏輯高）
